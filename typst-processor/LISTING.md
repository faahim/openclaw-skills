## Tagline

## Description

Tired of wrestling with Word formatting or LaTeX complexity? Typst is the modern document compiler that turns simple markup into beautiful PDFs in milliseconds.

This skill installs Typst automatically and provides ready-to-use templates for the documents you actually need: resumes, invoices, business letters, and reports. Write in clean, readable markup — get pixel-perfect output.

**What it does:**
- 🔧 Auto-installs Typst (Linux + macOS, x86 + ARM)
- 📝 Ready-to-use templates: resume, invoice, letter, report
- ⚡ Compile in <1 second (vs minutes for LaTeX)
- 📦 Batch compile entire directories
- 👁️ Watch mode — auto-recompile on save
- 🎨 Custom fonts, colors, and layouts
- 📤 Export to PDF, PNG, or SVG

**Who it's for:** Developers, freelancers, and anyone who needs professional documents without the overhead of Word or LaTeX.

## Core Capabilities

1. One-command install — auto-detects OS and architecture
2. Resume generator — professional CV from command-line options
3. Invoice generator — itemized invoices with auto-calculated totals
4. Letter generator — formal business letters from templates
5. Report generator — multi-section reports with table of contents
6. Batch compilation — compile all .typ files in a directory
7. Watch mode — live recompile on file changes
8. Multiple export formats — PDF, PNG, SVG
9. Community packages — access Typst's package ecosystem
10. Custom templates — create and reuse your own designs
