## Tagline

## Description

Building a frontend but the backend isn't ready yet? Need predictable API responses for testing? Tired of setting up Express routes just to mock some data?

Mock API Server spins up fully functional REST APIs from simple JSON files. Define your data structure, run one command, and get a complete API with CRUD operations, filtering, sorting, pagination, and relationships — no code required. It also supports OpenAPI/Swagger specs via Prism for schema-accurate mocking.

**What you get:**
- 🚀 Full REST API from a JSON file (GET, POST, PUT, PATCH, DELETE)
- 🔍 Built-in filtering, pagination, sorting, and full-text search
- 📄 OpenAPI/Swagger mock server with realistic fake data
- 📦 Pre-built scaffolds: ecommerce, blog, social, project management
- 🎲 Mock data generator (names, emails, custom fields)
- 🔄 CORS enabled, URL rewriting, response delays
- 👻 Daemon mode — run multiple mock servers simultaneously

Perfect for frontend developers, QA engineers, and anyone who needs a quick API without the overhead.

## Core Capabilities

1. JSON to REST API — Drop in a JSON file, get full CRUD endpoints
2. OpenAPI mocking — Generate realistic responses from Swagger/OpenAPI specs
3. Pre-built templates — Ecommerce, blog, social, project management scaffolds
4. Mock data generation — Create N records with typed fields (name, email, int, enum)
5. Filtering & search — Query parameters for filtering, sorting, pagination, full-text
6. URL rewriting — Custom route mapping (e.g., /api/v1/* → /*)
7. Response delays — Simulate slow networks with configurable latency
8. Multi-server — Run multiple mock APIs on different ports simultaneously
9. Daemon mode — Background operation with PID management and logging
10. Zero config CORS — Works with any frontend dev server out of the box
