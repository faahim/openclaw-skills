## Description

Formatting resumes by hand is tedious. Every time you update a job or tweak a bullet point, you're wrestling with Word docs or LaTeX templates. And tailoring resumes for different roles? Forget it.

Resume Builder lets you write your experience once in clean YAML, then generate polished PDF, HTML, and Markdown resumes with a single command. Three templates included (modern, classic, compact). Filter by sections or skills to create tailored versions for different applications. Even generates matching cover letters.

**What it does:**
- 📄 Generate PDF resumes with professional LaTeX typography
- 🌐 Generate responsive HTML resumes for your portfolio site
- ✍️ Generate clean Markdown for pasting into applications
- 🎨 3 templates: modern (minimal), classic (traditional), compact (single-page)
- 🎯 Filter sections and skills for tailored applications
- ✉️ Auto-generate matching cover letters
- 👀 Watch mode — auto-rebuild when you edit your YAML
- ⚡ 3-second builds from YAML to PDF

Perfect for developers, designers, and anyone who wants beautiful resumes without the formatting headache.

## Core Capabilities

1. YAML-to-PDF conversion — Professional LaTeX-rendered resumes in seconds
2. Multi-format output — PDF, HTML, and Markdown from one source file
3. Three templates — Modern, Classic, Compact for different contexts
4. Section filtering — Include only relevant sections per application
5. Skill filtering — Highlight specific skills for targeted roles
6. Cover letter generation — Matching cover letters from your resume data
7. Watch mode — Auto-rebuild on file changes during editing
8. Custom LaTeX headers — Extend templates with your own styling
9. Paper size options — US Letter or A4 support
10. XeLaTeX support — Full Unicode and custom font rendering
