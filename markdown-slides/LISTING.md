## Description

Building slide decks in PowerPoint or Google Slides is slow, repetitive, and impossible to version control. Every formatting tweak wastes time you could spend on content. You need a way to write slides as fast as you write docs.

Markdown Slides converts plain Markdown files into polished presentations using Marp CLI. Write your content with simple Markdown syntax, add slide separators (`---`), and export to HTML, PDF, or PowerPoint. Includes themes, speaker notes, background images, multi-column layouts, and custom CSS styling.

**What it does:**
- 📝 Write slides in Markdown — no GUI needed
- 📄 Export to HTML, PDF, or PPTX with one command
- 🎨 Three built-in themes + custom CSS support
- 🖼️ Background images, split layouts, diagrams
- 🗣️ Speaker notes via HTML comments
- 👁️ Live preview with hot-reload watch mode
- 📁 Batch convert entire directories of slide decks
- 🔧 5-minute setup — just needs Node.js

## Core Capabilities

1. Markdown to HTML — Browser-ready presentations with animations
2. Markdown to PDF — Share-ready documents, pixel-perfect
3. Markdown to PPTX — PowerPoint for corporate environments
4. Custom themes — CSS-based theming, reusable across decks
5. Live preview — Watch mode with auto-reload on save
6. Background images — Full, split-left, split-right layouts
7. Speaker notes — Hidden notes via HTML comments
8. Batch export — Convert all .md files in a directory
9. Multi-format — Export to all formats in one command
10. Version control — Plain text slides work perfectly with git
