# Listing Copy: SSH Bastion Manager

- Type: Skill
- Categories: security, dev-tools
- Price: $18
- Tagline: Harden SSH jump servers in minutes with audit-ready defaults

SSH Bastion Manager turns a plain Linux host into a secure jump box with key-only SSH, root login disabled, UFW firewall policy, fail2ban protection, and repeatable user onboarding scripts. It is built for developers, ops engineers, and founders who need secure remote access without hand-editing brittle config files.

What you get:
- Automated SSH hardening with backup + config validation
- Firewall setup for custom SSH ports and optional CIDR restrictions
- Fail2ban jail configuration for brute-force protection
- User provisioning with SSH public keys
- A one-command audit script to verify hardening state

Setup time: ~5-10 minutes on Debian/Ubuntu.
